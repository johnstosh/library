    Book Photos

    Import Book Photos from Google Photos

    This feature imports photos from your Google Photos account to automatically detect and create card catalog entries.

    How it works (Two-Phase Process):

    1. Phase 1 - Fetch Photos: Imports photos from Google Photos and saves them to the database
    2. Phase 2 - Process with AI: Uses AI to extract book information from photos and create book entries

    Note: Make sure you have configured your Google Photos API key in Settings.


    Photo Export Status

    This page shows the export status of all photos in the library to Google Photos.

    How it works:

    • Photos are exported to Google Photos
    • An album is created to hold the photos, named after this library
    • The permanent ID from Google Photos is stored in the database to keep track of export status
    • You can manually trigger an export for all pending photos or just individual photos
